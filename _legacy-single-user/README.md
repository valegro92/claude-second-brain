# Vault legacy — edizione solista (single-user)

Questa cartella conserva il vault dell'edizione precedente del template
(edizione solista, scritta intorno al profilo "Anna HR freelance").

## Cos'era

Il template originale: 1 utente, 4 layer di memoria, 3 regole non-negoziabili,
struttura `progetti/` flat. Pensato per un consulente o freelance che lavora
da solo con Claude.

## Perché è archiviato

Lo Step 1 della roadmap product-v1 ha sostituito il vault solista con un
vault PMI multi-utente (vedi `vault/` nella radice del repo): 6 layer di
memoria, 4 ruoli (Owner / Custode / Editor / Contributor), 8 pattern di
scrittura, regola dei 5 file per oggetto (clienti / fornitori / commesse).

Il vecchio modello non è più mantenuto: tutte le issue di audit legate al
vault legacy si risolvono qui, perché questa cartella è congelata.

## Quando

Archiviato il 2026-05-23 (Step 1 product-v1).

## Per la versione vivente

Vedi `vault/` nella radice del repo e `docs/06-framework-pmi.md` per il
modello attuale.
